# Assets Guide

Use this folder for project media.

Recommended structure:

- `backgrounds/` for full-page still images
- `characters/` for Dweller portraits or animated clips
- `video/` for looping MP4 backgrounds or scene inserts
- `audio/` for music and ambient sound

Where each type fits in the prototype:

- Background image: apply in `src/style.css`
- Background video: place in `index.html` behind the main interface
- Character image or MP4: place in `index.html` inside the contact profile panel

Path reminder:

- HTML uses `./assets/...`
- CSS uses `../assets/...`
